# Contributing

Thanks for wanting to improve BrowserUse.

## Setup

```bash
git clone https://github.com/angdwww/BrowserUse-VibeChatPOC.git
cd BrowserUse-VibeChatPOC
npm install
npm run setup:browsers
npm run check
npm run smoke
npm run start
```

## Making changes

1. Fork or branch the repo.
2. Make a focused change.
3. Run:

```bash
npm run check
npm run smoke
```

4. Open a pull request with a short explanation of what changed.

## Good first contributions

- Improve README examples.
- Add troubleshooting notes.
- Add more smoke-test cases.
- Improve terminal compatibility notes.
- Create a demo GIF or short video.
- Improve shot-mode UI details.

## Development style

Keep changes practical, small, and easy to test. BrowserUse is a proof of concept for the VibeChat development workflow, so improvements that make the demo clearer, easier to run, or easier to understand are especially useful.
