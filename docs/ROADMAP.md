# Roadmap

## v0.1 — Proof-of-concept foundation

- Terminal browser TUI.
- Playwright-backed navigation and interaction.
- Numbered links, buttons, and inputs.
- Context view.
- Shot/screenshot mode.
- Smoke test command.
- README and screenshots.

## v0.2 — Better developer experience

- More reliable shot-mode calibration controls.
- More setup and troubleshooting examples.
- Demo GIF or short video.
- Expanded smoke tests.
- Config file for default browser and terminal preferences.

## v0.3 — Stronger automation workflow

- More command chaining examples.
- Saved command recipes.
- Better page inspection helpers.
- More keyboard and mouse automation shortcuts.
- Cleaner plugin-style command organization.
