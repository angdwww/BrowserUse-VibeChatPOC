# Launch Posts

## One-line pitch

BrowserUse is a Playwright-powered terminal browser that lets you inspect and control a real web page from the CLI.

## GitHub post

```text
I built BrowserUse, an open-source terminal browser proof of concept powered by Playwright.

Why I made it:
Browser automation usually forces you to choose between a full visible browser or invisible scripts. I wanted a middle path where a real browser could still be navigated and inspected from a terminal UI.

What it does:
- loads real pages with Playwright
- shows page context and mapped controls in a terminal TUI
- supports commands for navigation, clicking, typing, scrolling, screenshots, and shot mode
- acts as a real proof of concept built through the VibeChat local development workflow

GitHub: https://github.com/angdwww/BrowserUse-VibeChatPOC
Built with: https://github.com/angdwww/VibeChat
Feedback welcome.
```

I wanted to see how far a real browser workflow could go inside a terminal UI without falling back to a normal browser window all the time.
