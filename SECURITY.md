# Security Policy

BrowserUse controls a real browser session through Playwright, so treat it like a local developer tool.

## Reporting security issues

Open a private report or contact the repository owner directly before posting sensitive details publicly.

## Safety notes

- Do not commit browser profiles, cookies, screenshots with private data, `.env` files, or local machine state.
- Review generated screenshots before sharing them.
- Use test sites or non-sensitive pages when recording demos.
- Keep the repo private while testing sensitive workflows.
